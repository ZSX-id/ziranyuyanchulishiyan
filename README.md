# 多模态医疗问诊系统

## 项目概述

本项目是一个基于阿里云百炼API的多模态医疗问诊系统，结合了高级RAG（检索增强生成）技术，能够处理复杂的医学文档和医学影像，为用户提供专业的医疗咨询服务。

## 功能特点

- **多模态支持**：支持文本、PDF和图像输入
- **高级RAG架构**：实现了混合检索（关键词检索）和重排序逻辑
- **长效记忆机制**：存储用户的历史提问和回答，实现跨会话的长期记忆
- **本地存储**：API凭证安全存储在本地文件中
- **多界面支持**：提供Streamlit和命令行两种界面

## 技术栈

- Python 3.7+
- Streamlit（Web界面）
- Requests（API调用）
- EasyOCR（图像识别，可选）
- PyPDF2（PDF处理，可选）

## 安装依赖

### 基本依赖
```bash
pip install streamlit requests
```

### 可选依赖（用于多模态支持）
```bash
pip install easyocr==1.4.2 PyPDF2
```

## 使用方法

### 1. Streamlit界面

1. 启动Streamlit应用：
   ```bash
   streamlit run streamlit_interface.py
   ```

2. 在浏览器中访问：http://localhost:8501

3. 在侧边栏输入阿里云API Key并保存

4. 上传医学文档（支持txt、pdf、jpg、png格式）

5. 在聊天界面输入问题，系统会基于上传的文档和历史记录生成回答

### 2. 命令行界面

1. 运行命令行应用：
   ```bash
   python cli_interface.py
   ```

2. 输入阿里云API Key（首次使用）

3. 开始提问，输入'exit'退出系统，输入'upload'上传文档

## 系统架构

### 核心组件

1. **MultimodalRAGSystem**：核心RAG系统，负责文档处理、检索和生成回答
2. **Streamlit Interface**：Web界面，支持文件上传和聊天功能
3. **CLI Interface**：命令行界面，支持基本的交互功能

### 工作流程

1. **文档处理**：系统会处理上传的文档，提取文本内容（对于图像会使用OCR）
2. **检索**：根据用户的问题，从文档库中检索相关信息
3. **生成**：使用阿里云百炼API生成专业的医疗回答
4. **记忆**：存储用户的交互历史，用于后续回答

## 阿里云API配置

1. 登录阿里云控制台，获取API Key
2. 在系统中输入API Key，系统会自动保存到本地文件
3. 后续使用时，系统会自动加载已保存的API Key

## 注意事项

- 本系统仅提供医疗咨询，不能替代专业医生的诊断和治疗
- 上传的医学文档会被处理并存储在内存中，系统退出后会丢失
- API Key存储在本地文件中，请确保文件安全

## 故障排除

### 常见错误

1. **API Key错误**：请检查API Key是否正确
2. **文件上传失败**：请确保文件格式正确且可访问
3. **OCR/PDF支持**：如果遇到OCR或PDF处理错误，请确保已安装相应的依赖

### 性能优化

- 对于大型文档，系统会自动分块处理
- 记忆系统会自动保留最近的10次交互，避免内存占用过高
